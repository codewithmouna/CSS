# 🎨 CSS Fundamentals

> A complete beginner-friendly journey through CSS fundamentals with examples, practice projects, and styling concepts.

# 🚀 What is CSS?

CSS (**Cascading Style Sheets**) is used to style and design web pages.

It helps to:

- 🎨 Add colors
- 📐 Create layouts
- ✨ Add animations
- 📱 Build responsive websites
- 🖌️ Improve UI design

## 🎯 2. Types of CSS

### 🟢 Inline CSS
Style directly inside HTML elements.

### 🔵 Internal CSS
Styles inside `<style>` tag.

### 🟣 External CSS
Separate `.css` file linked to HTML.
